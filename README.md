## About the app

This is a food delivery app created to offer you delivering food for you

It allows you to order any item from restaurants we offer and get it on your desired location

You first need to register to use the app.  For registration and log in there is used FireBase.

This application was originally created by Shalva Lomtadze and Giga Maisuradze

## Features

The android app lets you:
- order food from any location.
- order items from varius restaurants.
- search to find your favourite restaurant.
- chatbot to offer you suggestions about food.
- realtime chat with courier
- realtime update of courier location
- messaging and order notification updates
- dark mode and langueage change

## Permissions

Appliacation requires the following permissions:
- Full Network Access.
- Access to change wallpaper of your phone
- Access to your location
- Access to send notifications
